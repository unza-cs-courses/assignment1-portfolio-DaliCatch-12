## Project Overview
This project is a responsive portfolio website developed as part of Assignment 1 for CSC4035. The website showcases my skills, projects, and contact information using modern web development techniques. It demonstrates the use of semantic HTML5 structure, CSS3 styling, responsive layouts, and accessibility best practices.

The portfolio is designed using a mobile-first approach and adapts to different screen sizes including mobile, tablet, and desktop devices.

## CSS Techniques Used

The following CSS techniques were implemented in this project:

- CSS Custom Properties (Variables)
- Flexbox
- CSS Grid
- Media Queries
- Responsive Design
- Hover Effects and Transitions

Flexbox was used for layout elements such as the navigation bar, hero section, contact section, and footer.

CSS Grid was used to structure the project cards layout and create responsive multi-column layouts.

---

## Responsive Design

The website was designed using a **mobile-first approach** and includes the following breakpoints:

- Mobile: Default styles
- Tablet: `@media (min-width: 768px)`
- Desktop: `@media (min-width: 1024px)`
- Large Desktop: `@media (min-width: 1200px)`

These breakpoints ensure that the layout adapts smoothly to different screen sizes without horizontal scrolling.

---

## Challenges and Solutions

One of the main challenges during this assignment was ensuring that the layout adapted correctly across different screen sizes. This was solved by applying a mobile-first design strategy and using media queries to progressively enhance the layout for larger screens.

Another challenge was organizing the CSS so that it remained clean and maintainable. This was addressed by using CSS variables for colors, spacing, and typography, which made the design consistent and easier to update.
